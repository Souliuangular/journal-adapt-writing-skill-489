## Style Card

| Dimension | Observed writing move |
|-----------|-----------------------|
| Abstract | Connects welfare concept, policy instrument, empirical application, and implication. |
| Introduction | Frames equity as a design criterion, then narrows to how that criterion enters policy evaluation. |
| Gap | The gap is a missing welfare dimension, not merely a missing dataset. |
| Contribution | Contribution combines conceptual welfare clarification with empirical implementation. |
| Method exposition | Defines welfare and equity concepts before the application details. |
| Results style | Results are presented as welfare or policy-design comparisons. |
| Discussion | Uses empirical application to discipline policy interpretation. |
| Language | Formal economics register with explicit conceptual definitions. |

## Useful Rule Extracted

When a paper combines theory and data, JEEM-style writing should make the theoretical object legible before presenting empirical detail.
